Version: 5.3.4.1

Welcome to the AMPS C++ Client distribution!

This page provides links to documentation to help you get started developing applicaitons for AMPS, the Advanced Message Processing System. Within the C++ client distribution, the samples directory contains a set of samples and an AMPS configuration file that configures an AMPS server to support the functionality in the samples.

The distribution also includes full source code for the client, to help with debugging your applications and as a sample implementation for the extension points in the client.

For the latest documentation, client distribution, and server distribution, visit the 60East web site at http://crankuptheamps.com.
